# Password Generator
Lightweight password generator built with Python. 
Designed to output and store string of random characters.

## Features
- Password Generator
- Save-Search-Storage Functions
- Minimalist, sanitized UI
- Built with Python and related libraries

## Tech Stack
- **Frontend/Backend:** Python

## Getting Started

### Access
```bash
cd password-generator
python3 passgen.py
```

Follow instructions in terminal
