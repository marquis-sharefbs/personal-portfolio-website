## 🔒 Password Generator App

Located in `password-generator/`

This project is a lightweight local password generator app.

### Features

* Password Generator
* Save-Search-Storage Functions
* Minimalist, sanitized UI
* Built with Python and related libraries

### Running locally

From the project root:

```bash
cd password-generator
python3 passgen.py
```

Follow instructions in terminal

## 🌐 Personal Portfolio Website

Located in `personal-portfolio-website/`

This project is a personal website designed to showcase my work, skills, and projects.

### Features

* Responsive design
* Project showcase section
* Clean and minimal UI
* Built with modern web technologies

### Running locally

From the project root:

```bash
cd personal-portfolio-website
python3 -m http.server 8000
```

Then open:

```
http://localhost:8000
```
